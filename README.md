Juego de arcade (pacman) implementando particularidades, de difucultad.
 - Fantasma que se teletransporta
 - Invasión enemigos de otros juegos (DK y Goomba)
 - Aceleración progresiva movimiento de fantas
Implementación de librería pseudoaleatoria (cuadrado medio) hecha por este servidor,
que es la que hace el movimiento aleatorio de los fantasmas y otras funciones en el
juego.
También el uso de mapas distintos (3) como otro factor diferencial del juego original,
cambio de boosters frutas por bichos que cumplen la misma función, pero respetando esa
metodología de arcade (insert coin para poder jugar) y mostrando ranking de
puntuaciones y nivel.
